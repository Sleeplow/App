🚀 Premiers pas

Storage est une application d'inventaire qui vous aide à savoir ce que vous avez et où vous l'avez rangé.

  1. Ouvrez l'application — aucun compte requis.
  2. Créez votre première boite avec le bouton « + » en bas à droite de l'onglet Boites.
  3. Donnez-lui un numéro, un nom et une couleur, puis ajoutez vos articles.
Vos données se synchronisent automatiquement sur tous vos appareils Apple via iCloud — aucune configuration nécessaire.

📦 Boites

Une boite représente un contenant physique (boite en carton, bac, tiroir, etc.).

  • Numéro — identifiant visuel affiché sur la boite (ex. #1, #42).
  • Nom — description courte (ex. « Décorations Noël »).
  • Localisation — endroit où la boite est rangée (ex. Cave, Grenier). Des suggestions rapides s'affichent sous le champ pour aller plus vite.
  • Couleur — code couleur pour grouper vos boites visuellement.
  • Notes — informations complémentaires libres.
  • Localisation GPS — épinglez la boite sur une carte pour retrouver son emplacement précis.

Modifier ou supprimer une boite

Depuis la liste des boites, faites glisser une ligne vers la gauche pour accéder aux actions rapides (Modifier, Supprimer). Vous pouvez aussi modifier une boite depuis la vue détaillée via le menu « ⋯ » en haut à droite.

Trier les boites

Depuis l'onglet Boites, le menu « ⋯ » vous permet de trier par numéro, nom, couleur ou nombre d'articles.

🗃️ Articles

Un article est un objet rangé dans une boite. Pour en ajouter un, ouvrez une boite puis appuyez sur le bouton « + » en bas de l'écran.

  • Nom — désignation de l'article (obligatoire).
  • Descriptif — détails optionnels (couleur, taille, référence…).
  • Quantité — nombre d'exemplaires de cet article dans la boite.
  • Photo — prenez une photo ou choisissez-en une depuis votre bibliothèque pour identifier l'article visuellement.

Modifier ou supprimer un article

Depuis la vue détaillée d'une boite, faites glisser un article vers la gauche pour le modifier ou le supprimer.

Trier les articles

Dans le menu « ⋯ » de la vue détaillée, vous pouvez trier les articles par date d'ajout (anciens ou récents en premier), par nom ou par quantité.

🔍 Recherche

L'onglet Recherche vous permet de trouver n'importe quel article ou boite en quelques secondes. La recherche porte simultanément sur :

  • Le nom et le descriptif des articles
  • Le nom, le numéro et la localisation des boites

Les résultats s'affichent en deux sections — Boites et Articles — et se mettent à jour en temps réel à chaque caractère saisi.

📍 Localisation GPS

Chaque boite peut être épinglée sur une carte pour retrouver son emplacement physique précis (dans un entrepôt, un garde-meuble, plusieurs adresses…).

Épingler une boite

  1. Créez ou modifiez une boite.
  2. Dans la section « Localisation GPS », appuyez sur « Épingler sur la carte ».
  3. La carte s'ouvre centrée sur votre position actuelle. Faites glisser la carte pour positionner le repère rouge à l'endroit exact.
  4. Appuyez sur « Confirmer » pour valider.

Voir l'emplacement

Une miniature de carte apparaît dans la vue détaillée de la boite. Appuyez dessus pour ouvrir la carte en plein écran. Depuis cette vue, le bouton « Ouvrir dans Plans » lance l'application Plans d'Apple avec l'itinéraire vers la boite.

L'application demande l'accès à votre position uniquement lorsque vous choisissez d'épingler une boite. Vous pouvez révoquer cet accès à tout moment dans Réglages → Confidentialité → Service de localisation → Storage.

Supprimer un emplacement GPS

Dans le formulaire de modification d'une boite, appuyez sur « Supprimer l'emplacement » (en rouge).

🔲 Widget

Storage propose un widget pour l'écran d'accueil ou la vue Aujourd'hui qui affiche en un coup d'œil le nombre de boites et d'articles enregistrés.

Ajouter le widget

  1. Appuyez longuement sur l'écran d'accueil pour passer en mode édition.
  2. Appuyez sur « + » en haut à gauche.
  3. Recherchez « Storage » et sélectionnez la taille souhaitée.
  4. Appuyez sur « Ajouter un widget ».

Le widget se met à jour automatiquement dès que vous ajoutez ou modifiez des boites ou des articles dans l'application.

🔐 Verrou Face ID / Touch ID

Activez le verrou biométrique pour protéger l'accès à votre inventaire. Lorsqu'il est actif, l'application demande une authentification Face ID, Touch ID ou code à chaque ouverture.

Activer le verrou

Allez dans Réglages → Sécurité → Verrou Face ID et activez l'option.

Protection dans le sélecteur d'apps

Même lorsque l'application est en arrière-plan, un écran de confidentialité masque votre inventaire dans le sélecteur d'applications iOS.

⚙️ Réglages

L'onglet Réglages regroupe les préférences de l'application :

  • Verrou biométrique — activer / désactiver le verrou Face ID ou Touch ID.
  • À propos — version de l'application, liens vers l'aide et la politique de confidentialité.

❓ Questions fréquentes

Mes données sont-elles sauvegardées ?

Oui. Storage utilise iCloud (via CloudKit) pour synchroniser et sauvegarder vos données sur tous vos appareils Apple. Aucun compte supplémentaire n'est requis.

Comment récupérer des données supprimées ?

Les suppressions sont définitives. Si vous avez activé la Protection avancée des données iCloud, contactez le support Apple — la restauration de données iCloud peut être possible dans certains cas.

L'application fonctionne-t-elle sans Internet ?

Oui, toutes les données sont stockées localement sur votre appareil. La synchronisation iCloud reprend automatiquement dès que la connexion est rétablie.

Puis-je utiliser Storage sur plusieurs appareils ?

Oui. Connectez-vous avec le même identifiant Apple sur vos appareils iOS. L'application synchronise vos boites et articles automatiquement via iCloud.

Pourquoi l'emplacement GPS n'est-il pas disponible ?

Vérifiez que l'accès à la localisation est activé dans Réglages → Confidentialité → Service de localisation → Storage. Sélectionnez « Lors de l'utilisation ».

🚀 Getting Started

Storage is an inventory app that helps you know what you have and where you put it.

  1. Open the app — no account required.
  2. Create your first box with the "+" button at the bottom right of the Boxes tab.
  3. Give it a number, a name, and a color, then add your items.
Your data syncs automatically across all your Apple devices via iCloud — no setup required.

📦 Boxes

A box represents a physical container (cardboard box, bin, drawer, etc.).

  • Number — visual identifier displayed on the box (e.g. #1, #42).
  • Name — short description (e.g. "Christmas decorations").
  • Location — where the box is stored (e.g. Basement, Attic). Quick suggestions appear below the field for faster input.
  • Color — color-code to visually group your boxes.
  • Notes — free-form additional information.
  • GPS Location — pin the box on a map to find its exact physical location.

Edit or Delete a Box

From the box list, swipe a row left to access quick actions (Edit, Delete). You can also edit a box from the detail view via the "⋯" menu in the top right.

Sort Boxes

From the Boxes tab, the "⋯" menu lets you sort by number, name, color, or item count.

🗃️ Items

An item is an object stored inside a box. To add one, open a box then tap the "+" button at the bottom of the screen.

  • Name — item label (required).
  • Description — optional details (color, size, reference…).
  • Quantity — number of copies of this item in the box.
  • Photo — take a photo or pick one from your library to visually identify the item.

Edit or Delete an Item

From a box's detail view, swipe an item left to edit or delete it.

Sort Items

In the "⋯" menu of the detail view, you can sort items by date added (oldest or newest first), by name, or by quantity.

🔍 Search

The Search tab lets you find any item or box in seconds. The search covers simultaneously:

  • Item names and descriptions
  • Box names, numbers, and locations

Results appear in two sections — Boxes and Items — and update in real time with each character you type.

📍 GPS Location

Each box can be pinned on a map to record its exact physical location (warehouse, storage unit, multiple addresses…).

Pin a Box

  1. Create or edit a box.
  2. In the "GPS Location" section, tap "Pin on Map".
  3. The map opens centered on your current location. Drag the map to position the red pin exactly where you want.
  4. Tap "Confirm" to save.

View the Location

A map thumbnail appears in the box detail view. Tap it to open the full-screen map. From there, the "Open in Maps" button launches Apple Maps with directions to the box.

The app requests access to your location only when you choose to pin a box. You can revoke this access at any time in Settings → Privacy → Location Services → Storage.

Remove a GPS Location

In the box edit form, tap "Remove Location" (in red).

🔲 Widget

Storage offers a home screen or Today View widget that displays at a glance the number of boxes and items recorded.

Add the Widget

  1. Long-press the home screen to enter edit mode.
  2. Tap "+" in the top left.
  3. Search for "Storage" and select the desired size.
  4. Tap "Add Widget".

The widget updates automatically whenever you add or modify boxes or items in the app.

🔐 Face ID / Touch ID Lock

Enable biometric lock to protect access to your inventory. When active, the app requires Face ID, Touch ID, or passcode authentication each time it opens.

Enable the Lock

Go to Settings → Security → Face ID Lock and turn on the option.

Privacy in the App Switcher

Even when the app is in the background, a privacy screen hides your inventory in the iOS app switcher.

⚙️ Settings

The Settings tab contains app preferences:

  • Biometric lock — enable/disable Face ID or Touch ID lock.
  • About — app version, links to help and privacy policy.

❓ Frequently Asked Questions

Is my data backed up?

Yes. Storage uses iCloud (via CloudKit) to sync and back up your data across all your Apple devices. No additional account is required.

How do I recover deleted data?

Deletions are permanent. If you have iCloud Advanced Data Protection enabled, contact Apple Support — iCloud data restoration may be possible in some cases.

Does the app work without internet?

Yes, all data is stored locally on your device. iCloud sync resumes automatically once a connection is restored.

Can I use Storage on multiple devices?

Yes. Sign in with the same Apple ID on your iOS devices. The app syncs your boxes and items automatically via iCloud.

Why is GPS location unavailable?

Make sure location access is enabled in Settings → Privacy → Location Services → Storage. Select "While Using the App".
